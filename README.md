# SafeReport-AI

## 📌 Project Description
SafeReport-AI is an AI-based harassment reporting system that allows users to report incidents and analyze them using AI. It helps in identifying issues and ensuring safer environments.

## 🚀 Features
- User Login & Signup
- Report Submission System
- AI-based Analysis
- Location Detection
- Secure Data Storage

## ⚙️ Setup Instructions

1. Clone the repository:
   git clone https://github.com/your-username/SafeReport-AI

2. Install dependencies:
   cd backend
   npm install

   cd ../frontend
   npm install

3. Run the application:
   npm start

## 📊 Project Details
This project allows users to report incidents through a web interface. The backend processes the data and uses AI to analyze reports and provide insights.
